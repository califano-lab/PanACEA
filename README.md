The code is used for performing analysis in PanACEA. 

## Requirements

### System
- Python 3.8+
- R 4.0+ (required by `rpy2` and `pyreadr`)

Install R from [cran.r-project.org](https://cran.r-project.org) before installing Python dependencies.

### Python packages

```bash
pip install -r requirements.txt
```
